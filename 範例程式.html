範例程式

這一個範例程式直接使用 node REPL，也就是說，不需要寫一個檔案。

請去下載 node 最新版（5.8.0）。

  1. 建立一個資料夾，並 cd 進去。
  2. 下指令 npm install express@5.0.0-alpha.2 --save。
  3. 下指令 node 進入 REPL。（你也可以把它寫成檔案啦）
'use strict'
const express = require('express')
const app = express()
const router = app.router
app.get('/', (req, res, next) => { res.send(`你現在在 ${req.path} 路徑中`) })
router.get('/:id', (req, res, next) => { res.send(`你現在在路由器裡面，在 ${req.path} 路徑中，而且 id = ${req.params.id}`) })
app.use('/router/', router)
app.listen(3000)

這個範例非常簡單，而且沒有使用任何的中介軟體來處理要求的主體。

你可以測試以下網址：

http://localhost:3000/
http://localhost:3000/123
http://localhost:3000/router/123

理論上第二個應該要產生 404，不過因為 Express 5 在雷。
